参数控件视觉样式 + 存储结构

🎨 参数控件视觉样式

SVG预览中 float / int 控件的视觉表现

控件样式一览
$hole_dia float
🟦 float 控件（蓝色系）
$tooth_count int
🟩 int 控件（绿色系）
$angle float | 45.0
🟪 带默认值显示（紫色系 = 有预设值）
D5 未绑定
🟠 未绑定状态（橙色虚线）
实际SVG效果预览
视觉规则：
  • 🟦 float — 蓝色系标签（科技感，对应小数数值）
  • 🟩 int — 绿色系标签（自然感，对应整数数值）
  • 🟪 带默认值 — 紫色系，右侧显示默认数值
  • 🟠 未绑定 — 橙色虚线边框 + "???" 占位
  • 所有已绑定控件统一格式：$key: type

💾 存储结构设计

数据模型 + SVG文件格式 + 元数据

核心数据结构
// ===== 项目文件: drawing-001.parameaser ===== { "version": "1.0", "source": "drawing-001.dxf", "dimensions": [ { "id": "D1", "type": "aligned", // aligned | linear | radial | diameter | angular "value": 120.5, "unit": "mm", "center": { "x": 48.0, "y": 68.0 }, "rotation": -56.0, // 尺寸线旋转角度（度） "param": { // null = 未绑定 "key": "slant_length", "type": "float", // float | int "default": 120.5 } }, { "id": "D2", "type": "linear", "orientation": "horizontal", "value": 200.0, "unit": "mm", "center": { "x": 150.0, "y": 22.0 }, "param": null // 未绑定 } ] }
SVG文件结构
<svg> <!-- 2D图纸内容 (从DXF转换) --> <g id="drawing"> <!-- 所有图形实体 --> </g> <!-- 尺寸标注层 --> <g id="dimensions"> <!-- 尺寸线 + 文字 --> </g> <!-- 参数控件层 --> <g id="params"> <!-- float --> <rect class="param-float" x="..." y="..." /> <!-- int --> <rect class="param-int" .../> </g> <!-- SVG元数据 (隐藏) --> <metadata> <!-- JSON嵌入 --> </metadata> </svg>
文件系统
drawing-001/ ├── drawing-001.dxf ├── drawing-001.parameaser // 项目文件 (JSON) ├── drawing-001.svg // 输出SVG └── preview.html // 预览页 (可选)

📋 数据流汇总

① 解析 DXF → 提取所有尺寸标注实体（类型、值、中心坐标、旋转角度）

② 用户绑定 → 为每个尺寸选择 float/int + 输入 key → 写入 .parameaser 项目文件

③ 生成 SVG → 将 DXF图形 + 尺寸线 + 参数控件标签 合成为完整SVG

④ 预览/存储 → 可在浏览器预览SVG，也可导出对外使用
