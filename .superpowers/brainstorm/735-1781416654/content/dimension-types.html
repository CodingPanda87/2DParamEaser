四种尺寸标注类型 · 视觉方案

每种标注在SVG中的表现方式和参数控件绑定样式

📏 ① 对齐标注 (Aligned Dimension)

DXF图纸标记
SVG预览
特点：尺寸线与测量方向平行。控件标签沿尺寸线方向旋转，显示在尺寸文字位置附近。
中心坐标：尺寸文本的中心点。

📐 ② 线性标注 (Linear Dimension)

DXF图纸标记
SVG预览
特点：水平/垂直正交尺寸。控件标签水平显示在尺寸线上方，或垂直尺寸旁旋转90°。
中心坐标：尺寸文本的中心点。

⭕ ③ 半径/直径标注 (Radial / Diameter)

DXF图纸标记
SVG预览
特点：从圆心或弧心引出的标注线。控件标签放在标注线末端附近或尺寸值旁。
中心坐标：尺寸文本的中心点（对于半径，通常是标注文字位置）。

◤ ④ 角度标注 (Angular Dimension)

DXF图纸标记
SVG预览
特点：圆弧形式的标注线。控件标签放在圆弧内侧，靠近尺寸文本位置。
中心坐标：尺寸文本的中心点（沿圆弧方向）。

📊 控件标签样式规则

标注类型 控件位置 控件方向 控件内容
对齐标注 尺寸文字位置 沿尺寸线旋转 $key: type
线性标注 (水平) 尺寸线上方 水平 $key: type
线性标注 (垂直) 尺寸线右侧 垂直 (旋转90°) $key: type
半径/直径 标注线末端附近 水平 $key: type
角度标注 圆弧内侧 沿弧线方向或水平 $key: float
